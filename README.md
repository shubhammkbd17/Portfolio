## ✨ Key Features

- **Liquid Morphing Background**: A dynamic `canvas`-based liquid background animation that responds to user interaction and creates a deep, modern aesthetic.
- **Real-Time GitHub Stats Integration**: Connects directly to the GitHub API (`shubhammkbd17`) to fetch and display live statistics (repositories, stars, forks) and top programming languages using beautiful iconography. Fallback integration is built-in to handle API rate limiting seamlessly.
- **Embedded Live Projects**: My featured projects, including **TaskFlow** (MERN stack management app) and **Vgram** (CustomTkinter GUI), are embedded natively via interactive `iframes` allowing visitors to test the frontend directly from the portfolio.
- **Custom AI Chatbot**: An integrated `Ask AI` agent that is pre-trained on my personal knowledge base. It can answer visitor questions regarding my education, tech stack, and achievements intelligently.
- **Dynamic Typing Animation**: An eye-catching hero section featuring a custom-built typing and deleting animation effect.
- **Live Visitor Counter**: A floating visitor tracking mechanism simulating live interactions.

## 💻 Technology Stack

- **Structure**: HTML5
- **Styling**: Modern CSS3 (Vanilla)
  - Glassmorphism UI patterns
  - Flexbox & CSS Grid Layouts
  - Custom Keyframe Animations
  - Responsive Media Queries
- **Logic & Interactions**: Vanilla JavaScript
  - Intersection Observers for scroll-triggered animations
  - Asynchronous Fetch API for GitHub Integration
  - DOM Manipulation for Chatbot interface
- **Icons & Typography**: Font Awesome 6.0, Google Fonts (Poppins)
- **Form Handling**: EmailJS API for functional "Get in Touch" contact forms.

## 📂 Project Structure

- `index.html` - The core structure, semantic layout, and embedded iframes.
- `styles.css` - Global design system, typography, and interactive hover effects.
- `script.js` - UI logic, typing animations, scroll observers, and initialization.
- `github-real.js` - The asynchronous engine that fetches and renders GitHub profile statistics.
- `ai-agent.js` - The logic and knowledge base powering the floating AI assistant.
- `simple-email.js` - Integration script allowing seamless contact form submissions directly to `sybhu609@gmail.com`.
- `media/` - Local assets, profile images, and CV.
